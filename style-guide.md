# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Orange: hsl(25, 97%, 53%)

### Neutral

- White: hsl(0, 0%, 100%)
- Light Grey: hsl(217, 12%, 63%) -- circle color
- Medium Grey: hsl(216, 12%, 54%) -- selected state of circle
- Dark Blue: hsl(213, 19%, 18%) -- default circle gray
- Very Dark Blue: hsl(216, 12%, 8%) --

## Typography

### Body Copy

- Font size (paragraph): 15px

### Font

- Family: [Overpass](https://fonts.google.com/specimen/Overpass)
- Weights: 400, 700
